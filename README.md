### 1. Ace Superstore: Dimensional Modeling & Profitability Analysis

![Profitability Dashboard Screenshot](https://github.com/Tongai-Zinaka/RDAMP-Dimensional-Model-PowerBI/blob/main/visu%201.jpg?raw=true)

This project focused on building a robust, query-optimized reporting system for Ace Superstore. The primary goal was to transform a cleaned dataset into a structured **Star Schema dimensional model using SQL**. The resulting Power BI dashboard uncovered a critical, previously unseen profitability crisis, revealing that high revenue did not translate to profit.

*   **Skills:** Data Modeling (Star Schema), SQL (MySQL), Power BI, DAX, Profitability Analysis
*   **Tools:** MySQL, Power BI Desktop
